Ejercicios PAS

Realizados por José Pérez-Parras Toledano


  Ejercicio 1:
            Se hace la llamada junto a la carpeta que queremos compilar y como segundo argumento si queremos el nombre del comprimido


  Ejercicio 2:
            Se hace la llamada junto con el nombre del directorio sobre el que queremos generar el html

  Ejercicio 3:
            No se le llama junto con ningun argumento

  Ejercicio 4:
            Se hace la llamada junto con el nombre del fichero y el patrón que queremos encontrar

  Ejercicio 5:
            Se hace la llamada y como primer argumento un directorio y como segundo (opcional) un patron para buscar
            No ordena la salida

  Ejercicio 6:
            Se puede hacer la llamada sin ningun argumento y busca en la carpeta actual.
            O con varios argumentos y buscará en cada uno de los directorios pasados.
